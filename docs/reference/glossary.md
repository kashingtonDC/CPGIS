# GIS Glossary

Key terms and definitions for GIS.

## A

**Attribute** - Non-spatial information about a geographic feature
<br>**Azimuth** - Direction measured in degrees clockwise from north (0-360Â°)

## B

**Buffer** - Area of specified distance around features
<br>**Basemap** - Background reference map providing geographic context

## C

**CRS** - Coordinate Reference System
<br>**Clip** - Extract features from one layer that fall within boundary of another

## D

**Datum** - Reference framework for measuring locations on Earth
<br>**DEM** - Digital Elevation Model - raster representing terrain surface

## E

**EPSG** - European Petroleum Survey Group - maintains database of CRS codes
<br>**Extent** - Geographic area covered by a dataset

## F

**Feature** - Geographic object (point, line, or polygon) with attributes
<br>**Field** - Column in attribute table

## G

**Geocoding** - Converting addresses to geographic coordinates
<br>**Geodatabase** - Database optimized for storing geographic data
<br>**GeoJSON** - Geographic data format based on JSON
<br>**Geoprocessing** - Operations that transform geographic data
