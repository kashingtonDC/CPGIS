# Data Sources for Labs and Projects

## Free Global Data

### Natural Earth
**URL**: [naturalearthdata.com](https://www.naturalearthdata.com/)
<br>**Content**: Country boundaries, cities, physical features
<br>**Formats**: Shapefile, GeoPackage
<br>**Scale**: 1:10m, 1:50m, 1:110m

### OpenStreetMap
**URL**: [openstreetmap.org](https://www.openstreetmap.org/)
<br>**Content**: Roads, buildings, POIs, land use
<br>**Formats**: Various (use export tools)
<br>**Coverage**: Global, crowd-sourced

### USGS Earth Explorer
**URL**: [earthexplorer.usgs.gov](https://earthexplorer.usgs.gov/)
<br>**Content**: Satellite imagery, DEMs, aerial photos
<br>**Formats**: GeoTIFF, JPEG2000
<br>**Coverage**: Global (emphasis on US)

## US-Specific Data

### US Census Bureau
- **Boundaries**: States, counties, census tracts, blocks
- **Demographics**: Population, income, housing
- **URL**: [census.gov/geographies](https://www.census.gov/geographies/)

### USGS National Map
- **Content**: Topographic maps, elevation, land cover
- **URL**: [nationalmap.gov](https://www.nationalmap.gov/)
