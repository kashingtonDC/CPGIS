## Data Sources

### Global Data
- [Natural Earth](https://www.naturalearthdata.com/) - Free vector/raster map data

- [USGS Earth Explorer](https://earthexplorer.usgs.gov/) - Satellite imagery, elevation

## Tips for Self-Study

1. **Practice regularly** - Set aside consistent time each week
2. **Work with real data** - Use data relevant to your interests
3. **Join a community** - Connect with other learners
4. **Build a portfolio** - Document your projects
5. **Stay curious** - Explore topics beyond the basics
