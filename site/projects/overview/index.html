    Projects Overview

    Projects are hands-on assignments where you apply GIS skills to real-world problems, starting with conceptual thinking and progressing to technical mastery.

    Available Projects

    Project 0: Thinking Spatially

    Focus: Introduction to geographic thinking and map concepts

    Skills: - Spatial thinking across different domains - Map interpretation and analysis - Understanding USGS topographic maps - Exploring dynamic vs static maps - Project planning fundamentals

    Time: 2-3 hours

    Project 1: Mapping

    Focus: Map design and basic QGIS operations

    Skills: - Installing and using QGIS - Understanding map scales and projections - Creating professional maps - Working with attributes - Basic styling

    Time: 6-8 hours

    Project 2: Vector Data Analysis

    Focus: Vector operations and spatial analysis

    Skills: - Vector geometry types - Attribute data types - Tabular and spatial joins - Spatial queries - Data management

    Time: 8-10 hours

    Project 3: Raster Data Analysis

    Focus: Raster processing and terrain analysis

    Skills: - Raster vs vector concepts - Clipping and zonal statistics - Sampling raster values - Working with NoData - Creating heatmaps

    Time: 8-10 hours

    Project 4: Remote Sensing & Modeling

    Focus: Multi-criteria suitability analysis (GROUP PROJECT)

    Skills: - Remote sensing imagery - Multiple dataset integration - Criteria-based site selection - Professional presentations - Team collaboration

    Time: 12-15 hours (group)

    Final Project

    Focus: Independent research and analysis

    Skills: - Project design and scoping - Data acquisition - Complete GIS workflow - Professional reporting - Presentation (optional)

    Time: 20-25 hours over 5 weeks

    Project Philosophy

    Each project builds on previous work:

    1. Project 0: Develop spatial thinking foundations
    2. Project 1: Learn the software and basic concepts
    3. Project 2: Master vector analysis
    4. Project 3: Add raster analysis skills
    5. Project 4: Integrate multiple techniques (group)
    6. Final: Apply everything independently

    General Guidelines

    Before Starting

    • ✅ Read entire project description
    • ✅ Download all required data
    • ✅ Watch relevant tutorials
    • ✅ Plan your time

    During Work

    • ✅ Save frequently
    • ✅ Document your process
    • ✅ Take screenshots as you go
    • ✅ Ask questions early

    Before Submitting

    • ✅ Check all requirements met
    • ✅ Verify file formats
    • ✅ Include timestamps
    • ✅ Proofread written work

    Submission Requirements

    All projects require: - Written answers to questions - Screenshots with timestamps - Professional map outputs - Complete documentation

    File format: PDF preferred

    Grading

    Projects are graded on: - Completeness: All questions answered - Correctness: Accurate analysis - Quality: Professional presentation - Documentation: Clear methods

    Getting Help

    Resources: - Office hours (see syllabus) - Lab sessions (TAs available) - QGIS documentation - Tutorial websites - Classmates (discuss, but submit your own work)

    Best practices: - Start early - Work incrementally - Save your progress - Ask specific questions

    Tips for Success

    Time Management

    • Don't wait until the last minute
    • Break project into smaller tasks
    • Allow time for troubleshooting
    • Submit on time (late penalties apply)

    Technical Skills

    • Follow tutorials exactly
    • Save intermediate results
    • Check your work
    • Test on small datasets first

    Communication

    • Write clearly and professionally
    • Include all required elements
    • Explain your reasoning
    • Cite your sources
